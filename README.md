# Task Manager Project

## Project Description

This project was developed for the YZM202 Software Configuration course.

The system allows users to:
- create tasks
- complete tasks
- delete tasks
- list tasks

The project was designed using software engineering principles such as:
- Object-Oriented Programming
- Modularity
- Defensive Programming
- Unit Testing
- Integration Testing
- System Testing
- Refactoring
- Git version control

---

## Technologies Used

- Python
- pytest
- coverage
- Git & GitHub
